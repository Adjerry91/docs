VRCFaceTracking makes it easy to use your eye and or face tracking hardware of choice in VRChat.

By providing a standard of communcation to VRChat, users do not to worry about whether an avatar is compatible match their headset.

VRCFT uses a verstile module system for different face-tracking data sources, and makes adding new hardware support easy.

Ready to start using face tracking? Continue to the next section to [Get Started](./getting-started.mdx).
